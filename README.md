Libra
==========

Libra は、[SharpDX](http://sharpdx.org/) を基礎としたグラフィックス フレームワーク ライブラリです。DirectX 11 にのみ対応しています。

Libra は、一般的な利用を想定しておらず、willcraftia 個人が利用するためのライブラリとして、willcraftia が必要とする機能のみを実装します。つまり、一般的なグラフィクス ライブラリとして要求される機能であっても、willcraftia にとって不要な機能は実装対象外です。

Libra のクラス構造は XNA を参考にしていますが、Libra は XNA クローンではなく、また、XNA クローンを目指していません。

Libra は、これを利用するアプリケーションが SharpDX 非依存となるように設計しています。このため、SharpDX に既に存在する機能であったとしても、Libra で改めて実装している物があります（Matrix 構造体や BoundingBox 構造体など）。また、可能な場合には XNA へ類似させる方針であるため、SharpDX と同一の機能であっても実装が異なる場合があります。

現在、Libra におけるアセット ビルドでは、XNA コンテンツ プロジェクトの利用を前提としてます。すなわち、Libra の各サンプル アプリケーションは、XNA がインストールされた環境でなければビルドできません。なお、ビルド環境は XNA 依存ですが、実行環境は XNA 非依存です。

# 依存ライブラリ

Libra に含まれるプロジェクトをビルドするには、別途、外部ライブラリをビルド環境へ配備する必要があります。

必要となる外部ライブラリは以下の通りです。

+ SharpDX
+ [Musca](https://github.com/willcraftia/Musca) (ノイズ フレームワーク ライブラリ)
+ [Felis](https://github.com/willcraftia/Felis) (XNB デシリアライゼーション フレームワーク ライブラリ)

詳細は Libs ディレクトリ下に含まれる README を参照して下さい。

# サンプル アプリケーションに関する注意事項

Samples ディレクトリ下にある Libra サンプル アプリケーションは、Libra の動作確認のためのテスト アプリケーションでもあります。この動作確認のために、一部のサンプル アプリケーションは [XBOX LIVE indies games](http://xbox.create.msdn.com/ja-JP/) で提供されているサンプル アプリケーションの移植として構成しています。この場合、サンプル アプリケーションで利用するアセットは GitHub でのバージョン管理外であり、XBOX LIVE indies games より対応するサンプル アプリケーションをダウンロードし、そこに含まれるアセット ファイルを手動で配備する必要があります。
